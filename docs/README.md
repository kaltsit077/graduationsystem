## 🚀 快速访问

### 开发环境
- 启动开发环境：运行 `../scripts/start.bat`
- Docker 开发模式：运行 `../scripts/start-docker-dev.bat`

### 数据库操作
- 初始化数据库：运行 `../scripts/init-database.bat`
- 查看数据库操作指南：[database/DATABASE_OPERATIONS.md](database/DATABASE_OPERATIONS.md)

### 部署
- 快速部署：查看 [deployment/QUICK_DEPLOY.md](deployment/QUICK_DEPLOY.md)
- 生产部署：查看 [deployment/PRODUCTION_DEPLOY.md](deployment/PRODUCTION_DEPLOY.md)
